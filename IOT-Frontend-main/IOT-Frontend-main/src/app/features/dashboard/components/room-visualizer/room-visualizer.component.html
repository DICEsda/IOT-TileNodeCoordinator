3D Room Visualization

WASD to move • Mouse drag to look • Scroll to zoom (when not moving)

Avatar Controls:

  • W / ↑ - Move Forward
  • A / ← - Move Left
  • S / ↓ - Move Backward
  • D / → - Move Right

View Controls:

  • Mouse Drag - Look Around (while avatar controls active)
  • Mouse Drag - Rotate View (when no keys pressed, uses Orbit Controls)
  • Scroll - Zoom In/Out

Lighting: Lights turn ON when you're near, OFF when you move away.
